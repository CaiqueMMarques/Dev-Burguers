<h1 align="center">Dev-Burguers</h1>
<h2>Criei esse projeto para demonstrar oque aprendi sobre forEach, map, filter e reduce. A intenção foi fazer um pequeno cardápio com os 
produtos em preço normal, os produtos com desconto, o valor total dos produtos com o desconto, e filtrei apenas os produtos veganos.
<br>
<h2>Linguagens utilizadas</h2>

<p>-CSS</p>
<p>-JS</p>
